Bienvenu à l'Adrar

I- Présentation du centre

L'Adrar est un centre de formation contenant 5 pôles de formations que nous allons présenter ci-dessous.

1- Pôle numérique

Le pôle numérique de l'Adrar cible 3 types de formations : le développement web, les réseaux et la cybersécurité.
Pour la formation au Développement web, il s'agit ici d'apprendre à créer des sites web et des applications mobiles.
Il sollicite d'être formé au codage, HTML et CSS étant les langages de base.
L'Adrar propose une formation au PHP et SQL en plus des deux premiers, et JavaScript.
